# Receipts Product Spec

## Positioning
Receipts is a forensic-feeling self-insight app that turns messy life inputs into pattern cards, contradiction alerts, and weekly receipts.

## Core line
> Here's what your behavior says, not what your intentions say.

## Audience
- ambitious 20s/30s users
- productivity and self-improvement users
- founders and builders
- anxious overthinkers who want clarity

## MVP scope
### Inputs
- text notes
- mood score
- preset tags
- voice note upload + transcription

### Outputs
- pattern cards
- contradiction cards
- weekly receipt summary

## Design direction
- dark, premium, forensic
- black / charcoal base
- amber and soft red accents
- evidence-card UI

## Recommended stack
- Next.js
- TypeScript
- Tailwind CSS
- Supabase auth/database/storage
- OpenAI structured outputs

## Suggested launch plan
1. Launch a waitlist landing page.
2. Build capture + timeline.
3. Add structured insight generation.
4. Test with a small alpha group.
5. Charge after users specifically praise the insight quality.
