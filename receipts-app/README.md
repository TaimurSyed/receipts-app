# Receipts

Receipts is a dark, forensic-feeling self-insight app.

## Product idea
Users log messy life data — notes, moods, tags, and voice transcripts — and Receipts turns that into:
- pattern cards
- contradiction alerts
- weekly receipts

## Positioning
> Here's what your behavior says, not what your intentions say.

## MVP
- auth
- text entry capture
- mood + tags
- voice note transcription
- timeline
- AI-generated patterns, contradictions, and weekly receipts

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI / Anthropic for insight generation

## Getting started
```bash
npm install
npm run dev
```

## Notes
This scaffold currently focuses on the landing page and product direction. App flows, auth, and backend integration come next.
